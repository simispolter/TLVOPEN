# צינור הנתונים של פתוח לציבור

## מהו נתון GIS גולמי

נתון GIS גולמי הוא רשומה שהגיעה משכבת מידע עירונית או תכנונית. הרשומה יכולה לתאר פוליגון, מגרש, זיקת הנאה, הערת אזהרה, שימוש קרקע, תשתית או מידע טכני אחר. עצם קיומה של רשומה בשכבת GIS אינו אומר שהציבור רשאי להיכנס למקום או להשתמש בו.

## למה GIS גולמי אינו מרחב ציבורי מאומת

המטרה של הפרויקט היא לא להציג כל פוליגון עירוני. המטרה היא לזהות מקומות שבהם קיימת זכות ציבורית ממשית: מעבר, שהייה, כניסה, שימוש ציבורי או חובה תכנונית אחרת. לכן שכבת GIS היא מאגר מועמדים למחקר בלבד עד שיש ניסוח ציבורי ברור, מקור מאומת או בדיקת שטח.

## מושגים חשובים

שפ"פ: שטח פרטי פתוח. המונח לבדו אינו מוכיח גישה ציבורית. נדרש להבין את הוראות התוכנית או ההיתר.

זיקת הנאה: זכות שמוטלת על מקרקעין לטובת גורם אחר. זיקה לטובת העירייה או לתשתית אינה בהכרח זכות כניסה לציבור.

זכות עירונית או הערת אזהרה: יכולה להיות טכנית, תשתיתית או קניינית. היא אינה בהכרח זכות ציבורית.

זכות גישה ציבורית: ניסוח שמצביע על מעבר, שהייה, שימוש ציבורי, גישה חופשית או פתיחה לציבור. גם כאן נדרש אימות לפני ניסוח חד-משמעי.

## סטטוסי סיווג

`candidate_raw`: מועמד לבדיקה. רשומת מקור גולמית בלבד.

`candidate_likely_public`: כנראה פתוח לציבור - דורש אימות. נמצאו ביטויים שמרמזים על זכות ציבורית, אך אין עדיין אימות מלא.

`verified_public_access`: פתוח לציבור לפי מקור מאומת. רשומה ידנית או מקור רשמי מאומת תומכים בזכות הציבור.

`field_verified_open`: נבדק בשטח ונמצא פתוח.

`reported_blocked`: דווח כחסום. יוצג לציבור רק אם קיימת זכות ציבורית מאומתת או סבירה בבסיס.

`excluded`: לא רלוונטי לציבור. בדרך כלל רשומות תשתית, תחזוקה, רכב או זכות טכנית.

## מה מופיע במפה הציבורית

ברירת המחדל טוענת את `public/data/public/public-spaces.geojson`. היא כוללת רק רשומות מסוג:

- `verified_public_access`
- `field_verified_open`
- `reported_blocked`, רק אם קיימת זכות ציבורית בסיסית מאומתת או סבירה

רשומות ללא גיאומטריה מדויקת נשמרות במאגר, אבל אינן מוצגות כפוליגון על המפה.

## מה נשאר בשכבת המחקר

`public/data/research/research-candidates.geojson` כולל מועמדים לא מאומתים:

- `candidate_likely_public`
- `candidate_raw`

שכבה זו אינה רשימה של מרחבים פתוחים לציבור. היא מיועדת למחקר, בדיקת מקורות, הצלבה עם תוכניות והכנה לסיורי שטח.

## איך מריצים את הסקריפטים

```bash
npm run data:fetch:tlv
npm run data:classify
npm run data:build
```

או הכל יחד:

```bash
npm run data:update
```

הפקודות יוצרות או מעדכנות:

- `public/data/raw/tlv-easements-with-plan.geojson`
- `public/data/raw/tlv-easements-without-plan.geojson`
- `public/data/raw/tlv-landuse-detailed.geojson`
- `public/data/processed/tlv-public-access-candidates.geojson`
- `public/data/public/public-spaces.geojson`
- `public/data/research/research-candidates.geojson`

## מגבלות נוכחיות

המסווג מבוסס מילות מפתח ושמרנות תכנונית. הוא לא מחליף קריאת תוכנית, היתר בנייה, נסח, החלטת ועדה או בדיקת שטח.

שכבת שימושי הקרקע רחבה מאוד. הסקריפט סורק אותה בפניות מדורגות, אך שומר לקובץ העבודה רק רשומות שפ"פ או שטח פרטי פתוח שנמצאו בסינון המקומי, כדי לא להכניס למאגר הקוד קובץ ענק שאינו רלוונטי למחקר. אם שירות ArcGIS נכשל, הסקריפט שומר קבצים קיימים או יוצר קובץ ריק עם סיבת כשל.

רשומות ידניות מסוימות הן עוגני מחקר. כאשר אין מיקום מדויק, אין להמציא גיאומטריה.
