# פתוח לציבור / TLVOPEN

מפת עיתונות אזרחית לזיהוי מרחבים פרטיים שאמורים להיות נגישים לציבור: מעברים, אטריומים, רחבות, קולונדות, שפ"פים או מקומות עם חובת גישה ציבורית לפי מקור תכנוני או בדיקת שטח.

## סטטוס הפרויקט

זהו MVP סטטי ומקומי על בסיס Next.js, TypeScript, Tailwind CSS ו-MapLibre. אין עדיין בסיס נתונים, אימות משתמשים, Supabase או מערכת דיווח חיה.

הנתונים הגולמיים אינם מוצגים כברירת מחדל כמרחבים ציבוריים מאומתים. ברירת המחדל טוענת רק את המאגר הציבורי המסונן. שכבת המחקר הלא מאומתת זמינה דרך מתג ייעודי בממשק.

## אזהרת נתונים

שכבות GIS עירוניות הן מאגר מועמדים למחקר בלבד. זיקת הנאה, שפ"פ, הערת אזהרה או זכות לטובת העירייה אינם מוכיחים בפני עצמם שהציבור רשאי להיכנס. נדרש מקור תכנוני ברור, מקור מאומת, אוצרות ידנית או בדיקת שטח.

## התקנה

```bash
npm install
```

## פיתוח

```bash
npm run dev
npm run lint
npm run build
```

## פקודות נתונים

```bash
npm run data:fetch:tlv
npm run data:classify
npm run data:build
npm run data:update
```

הפקודות שומרות נתוני מקור תחת `public/data/raw`, מועמדים מסווגים תחת `public/data/processed`, שכבת מפה ציבורית תחת `public/data/public`, ושכבת מחקר תחת `public/data/research`.

## מבנה עיקרי

- `src/components/PublicMap.tsx` - מפת הבית והחלפת שכבת מחקר.
- `src/lib/classifySpace.ts` - מסווג שמרני לרשומות GIS.
- `src/lib/normalizeSpace.ts` - מודל `Space` אחיד.
- `scripts/fetch-tlv-layers.mjs` - הורדת שכבות ArcGIS רשמיות.
- `scripts/classify-tlv-candidates.mjs` - סיווג מועמדים.
- `scripts/build-public-datasets.mjs` - בניית מאגר ציבורי ומאגר מחקר.
- `docs/DATA_PIPELINE.md` - תיעוד מפורט של צינור הנתונים.

## מגבלות

המפה אינה משתמשת בשרתי אריחים ציבוריים של OpenStreetMap. כרגע היא מציגה רק רקע מקומי פשוט ופוליגונים מקומיים.

טופס הדיווח מקומי בלבד ואינו שומר מידע לשרת.

רשומות ללא מיקום מדויק נשמרות במאגר אך אינן מוצגות כפוליגון.
